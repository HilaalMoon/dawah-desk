Da'wah Desk — Installation Guide
==================================

Step 1 — Download and extract
-------------------------------
Download the latest version from:
  https://github.com/HilaalMoon/dawah-desk/releases/latest
On that page, under "Assets", click "Source code (zip)" to download.
Create a folder called "Dawah-Desk" anywhere on your computer.
Extract the contents of the zip into that folder.

Step 2 — Install
-----------------
Open the Dawah-Desk folder.
Double-click 00_INSTALL_DawahDesk.bat
A window will appear showing installation progress.

First time only — Node.js:
Da'wah Desk needs a free, standard tool called Node.js to run.
The very first time you install, if Node.js is not already on your computer,
the installer will open the Node.js download page and then stop.
If that happens:
  1. Install Node.js using the default options (just keep clicking Next, then Install)
  2. Run 00_INSTALL_DawahDesk.bat again
You only ever do this once. If Node.js is already installed, this is skipped.

The installer then automatically downloads the rest of the app's components.
This needs an internet connection and takes 1-2 minutes — no action needed from you.
Wait until a completion message appears.
A shortcut called "DawahDesk" will be added to your desktop.

Step 3 — Enter your credentials
---------------------------------
Da'wah Desk will open in your browser automatically.
If this is your first time, you will be asked to paste your credential file.
Paste the full contents of the credential file provided by your system administrator.
Click "Save and open Da'wah Desk".

Step 4 — Start using the app
------------------------------
Da'wah Desk is now ready to use.
To open it in future, double-click the DawahDesk shortcut on your desktop.
With AI credentials, you can use AI features across the app.
It is recommended to select "Google Vertex AI / Gemini 2.5 Flash (Vertex)" for faster results.

Updating to a new version
--------------------------
When a new version is available:
1. Download the latest version from:
   https://github.com/HilaalMoon/dawah-desk/releases/latest
   (under "Assets", click "Source code (zip)")
2. Create a new folder (e.g. "Dawah-Desk-v2") and extract the zip into it
3. Run 00_INSTALL_DawahDesk.bat from the new folder
4. Your cases, sources, and credentials are preserved automatically
5. Once confirmed working, you can delete the old folder

If anything goes wrong
-----------------------
Open the installation log file at:
C:\Users\[your username]\AppData\Local\Temp\DawahDesk_install_log.txt
Share this file with your system administrator for help.
